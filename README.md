# 🌿 Greenden Products Page

A responsive products page built using **HTML** and **Tailwind CSS**. This project showcases a modern and mobile-friendly design for displaying products with a clean user interface.

## 🚀 Features

- Responsive design for mobile, tablet, and desktop devices
- Modern UI built with Tailwind CSS
- Product card layout
- Clean and user-friendly design
- Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
